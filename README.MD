# AI Engineering Explained 🚀

Notes and code examples for **"AI Engineering"** by Chip Huyen.

## What's This?

Breaking down the book chapter-by-chapter with:
- 📝 Notes and summaries
- 💻 Working Python code
- 🎯 Key takeaways
- ⚠️ Common mistakes to avoid

## Chapters

| # | Chapter | Topics |
|---|---------|--------|
| 1 | Introduction to Building AI Applications with Foundation Models | AI engineering fundamentals, why it matters, new AI stack |
| 2 | Understanding Foundation Models | LLMs, multimodal models, how they work |
| 3 & 4 | Evaluation Methodology & Evaluating AI Systems | Metrics, offline/online evaluation, AI-as-judge |
| 5 | Prompt Engineering | Prompting techniques, best practices, system prompts |
| 6 | RAG and Agents | Retrieval-Augmented Generation, agent patterns, architectures |
| 7 | Finetuning | When to finetune, data prep, LoRA, parameter-efficient training |
| 8 | Dataset Engineering | Data collection, quality, deduplication, filtering |
| 9 | Inference Optimization | Quantization, batching, latency, cost optimization |
| 10 | AI Engineering Architecture and User Feedback | System design, feedback loops, monitoring, iteration |

## Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/ai-engineering-explained.git

# Read Chapter 1
cat chapters/01-introduction/README.md
```
